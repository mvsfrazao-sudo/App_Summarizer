Simple API Integration

RESTful API designed for easy integration into your existing workflows

📤 Upload Paper

POST /api/papers/upload Content-Type: multipart/form-data curl -X POST "your-domain.com/api/papers/upload" \ -F "file=@paper.pdf"

📊 Check Status

GET /api/papers/{paper_id}/status { "status": "completed", "progress": 100 }

📋 Get Summary

GET /api/papers/{paper_id}/summary { "title": "How AI Agents Are Revolutionizing Economic Research", "introduction": "Imagine having a research assistant...", "key_points": [...], "implications": [...] }

🚀 Quick Setup

Prerequisites

  • Node.js 18+ and Yarn
  • Python 3.11+
  • MongoDB
  • Emergent LLM Key

Environment Setup

# Backend .env MONGO_URL=mongodb://localhost:27017 EMERGENT_LLM_KEY=your-key # Frontend .env REACT_APP_BACKEND_URL=http://localhost:8001

Installation Commands

# Backend setup cd backend pip install -r requirements.txt uvicorn server:app --host 0.0.0.0 --port 8001 # Frontend setup cd frontend yarn install yarn start
